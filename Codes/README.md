## Codes
This folder contains code examples that correspond to the material covered in each section. These examples are designed to help you get hands-on experience with the concepts covered in the slides.

## Content

### Basics: 
* 1_LED_Blink
* 2_Serial
* 3_Push_Button
* 4_Potentiometer
* 5_LED_Fade

### Sensors:
* 1_LDR
* 2_Ultra_Sonic
* 3_DHT
* 4_IR

### Actuators:
* 1_DC_Motor
* 2_Servo_Motor
* 3_Buzzer

### Communication:
* 1_Serial_Monitor
* 2_Chat_Bluetooth
* 3_LED_Control

### Display:
* 1_LCD_Print
* 2_LCD_Clear
* 3_LCD_Blink
* 4_LCD_Scroll_for_Loop
* 5_LCD_Scroll_Function
* 6_SSD_Without_Library
* 7_SSD_With_Library
